JavaScript

JavaScript (JS) é uma linguagem leve, interpretada e baseada em objetos, amplamente usada para páginas web e outros ambientes como Node.js. É dinâmica, baseada em protótipos e suporta múltiplos paradigmas, incluindo orientação a objetos e programação funcional.

HTML5

HTML (Linguagem de Marcação de HiperTexto) é a base da web, definindo a estrutura e o significado do conteúdo. Ele trabalha junto com CSS para a apresentação e JavaScript para a funcionalidade.

Node.js

Node.js® é uma ambiente de execução de JavaScript disponível para várias plataformas, de código aberto e gratuita, que permite os programadores criar servidores, aplicações da Web, ferramentas de linha de comando e programas de automação de tarefas.
